# Notebooks de Análisis y Entrenamiento

Esta carpeta contiene los notebooks de Jupyter para cada modelo:

1. **01_Regresion_Logistica.ipynb** - Análisis y entrenamiento del modelo de Regresión Logística
2. **02_KNN.ipynb** - Análisis y entrenamiento del modelo K-Nearest Neighbors
3. **03_KMeans.ipynb** - Análisis y entrenamiento del modelo K-Means Clustering

Cada notebook debe incluir:
- Carga y exploración de datos
- Preprocesamiento
- Entrenamiento del modelo
- Evaluación y métricas
- Visualizaciones (gráficas)
- Guardado del modelo entrenado
